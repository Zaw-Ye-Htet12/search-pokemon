# Search Pokémon

A modern web application to search, view, and manage your favorite Pokémon. Built with Next.js, TypeScript, and GraphQL.

## Features

- Search and browse Pokémon
- View detailed Pokémon information
- Mark and manage favorite Pokémon
- Responsive and accessible UI
- Fast loading with skeleton screens

## Tech Stack

- [Next.js](https://nextjs.org/)
- [React](https://react.dev/)
- [TypeScript](https://www.typescriptlang.org/)
- [GraphQL](https://graphql.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Apollo](https://www.apollographql.com/docs/react) 

## Getting Started

1. Install dependencies:
      npm install
   

2. Run the development server:
      npm run dev
   

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- src/app/ – Application pages and layouts
- src/components/ – Reusable UI components
- src/graphql/ – GraphQL queries
- src/hooks/ – Custom React hooks
- src/interfaces/ – TypeScript interfaces and types
- src/config/ – App configuration and constants
- src/lib/ – Utility libraries
- src/providers/ – Context providers
- src/stores/ – State management

## Scripts

- npm run dev – Start development server
- npm run build – Build for production
- npm run start – Start production server
- npm run lint – Lint code
